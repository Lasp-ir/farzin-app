# لیست تسک‌های توسعه لابراتوار آنالیز (Analysis Board)

- [ ] **پشتیبانی از کیبورد (Keyboard Navigation):** اتصال کلیدهای جهت‌نمای چپ و راست برای پیمایش سریع در درخت حرکات.
- [ ] **اختلاف مهره‌ها (Material Imbalance):** محاسبه و نمایش گرافیکی مهره‌های برتری زیر اسم هر بازیکن.
- [ ] **توقف/اجرای موتور (Pause Engine):** اضافه کردن دکمه Pause/Play برای مدیریت مصرف CPU و باتری توسط Stockfish.
- [ ] **نمایش اسم گشایش (ECO / Opening Name):** تشخیص و نمایش زنده نام گشایش و واریانت‌ها در ابتدای بازی.
